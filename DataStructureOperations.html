DataStructureOperations.CombineSparseMatricesRowWise(MainMatrix, AddedMatrix)[source]¶

Stack two scipy sparse matrices vertically (row wise). Will initialize the main matrix to be two dimensional csr_matrix with all zero elements if the main matrix is empty.

Parameters:
  • MainMatrix (SparseMatrix) – The main matrix that you want to add the AddedMatrix.
  • AddedMatrix (SparseMatrix) – The matrix added followed by the main matrix.
Returns:

Result: The result of Stack sparse matrices vertically (row wise).
Return type:

SparseMatrix

DataStructureOperations.FlattenList(List)[source]¶

Flatten a list using itertools no matter how many nest it has. E.g. [[‘foo’, ‘baz’], [‘gg’]] or [[[‘foo’, ‘baz’], [‘gg’]]] to [‘foo’, ‘baz’, ‘gg’].

Parameters:List[Variant] – The list you want to flatten
Returns:List: Flattened list
